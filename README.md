# Marathon Club Management System

This is a Java desktop application for managing athletes and marathon clubs.

## Features
- Add, delete, and view athletes
- Calculate experience and find the most experienced athlete
- Filter athletes by minimum experience
- Group athletes by favorite distance

## Technologies
- Java (JVM)
- Collections (Map)
- Java Stream API

## How to Run
1. Compile all `.java` files in the `src` folder.
2. Run `Main.java`.
